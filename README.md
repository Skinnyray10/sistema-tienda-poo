Sistema de Gestión de Inventario (POO)
Un simulador de tienda de consola desarrollado en Python. Este proyecto modela un sistema de inventario utilizando Programación Orientada a Objetos (POO), aplicando herencia para separar productos tecnológicos y ropa, gestionando su stock de forma automática.

Características:

- Arquitectura POO: Uso de Clases, Objetos, y Herencia.

- Gestión de Stock: Método integrado que actualiza el inventario automáticamente tras cada venta.

- Categorización: Clases hijas especializadas (Electronico y Ropa) con atributos únicos como meses de garantía y talla.

- Principio DRY: Uso de super().__init__ para optimizar y no repetir código.
  
Tecnologías Usadas (Stack)
- Python 3.x
